Как я контролировал и фиксил:
- вручную проверял UI на разных экранах, тестировал адаптивность
- убирал фиксированные размеры (если были)
- использовал flexbox и добавлял SafeAreaView
- корректировал отступы через дополнительные промпты, ну и более расписанными их делал (опять же через ChatGPT, чем детальнее описана задача, тем лучше ИИ с ней справляется)
